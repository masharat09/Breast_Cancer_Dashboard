🎗️ **Breast Cancer Analysis Dashboard | Power BI**

📌 Project Overview

    The Breast Cancer Analysis Dashboard is an interactive Power BI dashboard developed to analyze breast cancer patient data. The dashboard provides insights into patient demographics, survival status, tumour stages, histology types, surgery trends, and age distribution. It enables healthcare professionals and analysts to explore the dataset through interactive visualizations and filters.

🎯 Objective

    The primary objective of this project is to:

    Analyze breast cancer patient records.
    Monitor patient survival and mortality rates.
    Understand tumour stage distribution.
    Study histology patterns.
    Analyze surgery trends over time.
    Provide an interactive dashboard for better decision-making.
    
🛠️ Tools & Technologies

    Power BI Desktop
    Power Query
    DAX (Data Analysis Expressions)
    
🧹 Data Cleaning

    The following data cleaning steps were performed in Power Query:

    Removed duplicate records
    Checked and handled missing values
    Corrected inconsistent data types
    Standardized categorical values
    Extracted Surgery Month and Surgery Year
    Created Age Group categories

📈 Key Insights

    Total Patients: 321
    Alive Patients: 255
    Dead Patients: 66
    Average Age: 58.88 Years
    Survival Rate: 79%
    Death Rate: 21%
    Tumour Stage II has the highest number of patients.
    Most patients belong to the 61+ age group.
    Infiltrating Ductal Carcinoma is the most common histology type.
    Surgery cases remain relatively consistent throughout the year with slight monthly variations.
